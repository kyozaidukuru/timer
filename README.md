# 残り時間がみえる学習タイマー

円弧で残り時間を表示する、直感的に使える学習用タイマーです。  
学習時間や作業時間を「見える化」することで、時間の感覚をつかみやすくします。

---

## 特徴

- 円弧で残り時間がひと目でわかる
- タッチ／クリックで時間をセット
- タイマーを「固定」して誤操作を防止
- 子どもから大人まで使える設計
- Webブラウザだけで動作（インストール不要）

---

## 使い方

1. 円盤の好きな時間（分）をクリック／タッチします  
2. タイマーがスタートし、残り時間が円弧で減っていきます  
3. もう一度時間を押すと、残り時間を変更できます  
4. 「固定」ボタンを押すと、時間変更ができなくなります  
5. もう一度押すと固定が解除されます
